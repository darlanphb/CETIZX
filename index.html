Projetos de Alunos

Destaques dos Cursos de Tecnologia

Reconhecimento de Imagens para Classificação

Módulo de Visão Computacional

Desenvolvimento de um modelo de Deep Learning (Redes Neurais Convolucionais) capaz de identificar e classificar objetos em tempo real. O projeto visa aplicações práticas na agricultura e segurança escolar.

  • Tecnologias: Python, TensorFlow/Keras.
  • Impacto: Aceleração da análise e tomada de decisão.

Chatbot de Suporte Educacional

Processamento de Linguagem Natural (PLN)

Criação de um assistente virtual baseado em regras e modelos de linguagem simples para responder dúvidas frequentes dos alunos sobre horários, notas e materiais didáticos da escola.

  • Tecnologias: JavaScript, APIs de NLP.
  • Impacto: Melhoria da comunicação interna da instituição.

Sistema de Gestão de Biblioteca (Web)

Projeto Full-Stack

Desenvolvimento de um sistema completo para controle de acervo, empréstimos e usuários. Utiliza arquitetura MVC, demonstrando proficiência em front-end, back-end e integração com banco de dados.

  • Tecnologias: HTML, CSS, JavaScript (Front-end), Node.js (Back-end).
  • Impacto: Projeto funcional para uso interno do CETI.

Aplicação Mobile de Monitoramento de Estudos

Desenvolvimento Mobile Básico

Protótipo de aplicativo para rastrear o tempo gasto em diferentes disciplinas, utilizando metodologias ágeis (Scrum) e princípios de Design UX/UI.

  • Tecnologias: Ferramentas de prototipagem (Figma), Lógica de programação.
  • Impacto: Desenvolvimento de habilidades de gestão de projetos.

Modelagem de Dados para E-commerce Local

Design e Normalização

Criação de um modelo Entidade-Relacionamento (ER) de alta complexidade e sua posterior normalização (até a 3FN), garantindo integridade, consistência e eficiência para um sistema de vendas online fictício.

  • Tecnologias: Diagramas ER/UML, Normalização.
  • Impacto: Base sólida para qualquer desenvolvimento de sistema.

Otimização de Consultas SQL

Performance e Índices

Estudo de caso focado na análise de planos de execução de consultas (EXPLAIN) e na criação de índices estratégicos para otimizar o tempo de resposta em grandes volumes de dados simulados.

  • Tecnologias: SQL (PostgreSQL/MySQL), Análise de performance.
  • Impacto: Foco na escalabilidade e manutenção de sistemas.
